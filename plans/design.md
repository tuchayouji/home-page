# 博客设计文档

## 基本信息
- **博客名：** Tucha 的博客
- **技术栈：** Astro + TailwindCSS
- **部署：** Vercel

## 设计风格
- 暖橙色系二次元风
- 首页参考格式：全屏背景图 + 头像 + 名字 + 时钟 + 导航(图标+文字) + 底部签名

## 导航结构
- 主页（首页）
- 文章 → 跳转独立页面
- 我的
  - 追番
  - 日记
  - 相册
- 关于
  - 关于我
  - 友链
- 其他
  - 项目展示
  - 技能展示
  - 时间线

## 首页布局
- 全屏背景图（用户后续提供，先用占位）
- 顶部：圆形头像（用户提供）
- 博客名 "Tucha 的博客"
- 数字时钟
- 导航按钮：主页 / 文章 / 我的 / 关于 / 其他（带下拉菜单）
- 导航样式：透明背景 + Lucide 图标 + 纯文字
- 下拉菜单：点击展开
- 底部：个性签名（用户后续添加）

## 各页面

### 文章页面
- 布局：主区域 + 右侧边栏
- 主区域：文章列表，一排一列，每篇包含：
  - 标题
  - 日期
  - 标签
  - 简介摘要
- 侧边栏内容：（待定）
- 页面顶部有归档导航/筛选
